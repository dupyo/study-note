# 워터마킹(Watermarking)

저작권자의 권리를 보호하기 위해 이미지나 영상에 저작권 정보를 삽입하는 기술을 말함

워터마킹 삽입 방법

- 주파수 영역 기술
- 공간 영역 기술

- 워터마크가 삽입된 미디어를 변형한다면 변형 여부와 변형된 위치를 검출할 수 있음
- 삽입된 워터마크를 제거하기 위한 필터링이나 왜곡과 같은 다양한 공격으로부터 저작권 정보를 훼손 없이 추출할 수 있어야 함
